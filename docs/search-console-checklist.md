# Search Console Checklist

Use this after pushing live SEO changes.

1. Open Google Search Console for `https://soultails.com/`.
2. Verify the preferred domain/property is correct.
3. Submit `https://soultails.com/sitemap.xml`.
4. Use URL Inspection on:
   `https://soultails.com/`
   `https://soultails.com/services`
   key service pages like `https://soultails.com/services/pet-transport`
5. Request indexing for the homepage and updated service pages.
6. Check that the live page source includes:
   favicon links
   `manifest.webmanifest`
   Organization/WebSite JSON-LD
7. In the Enhancements or Rich Results views, confirm Google can read the structured data.
8. In Page Indexing, make sure important public pages are not blocked by:
   `noindex`
   robots rules
   redirects
9. Recheck the homepage title and description snippet after Google recrawls.
10. If favicon/logo still show old results in Google, wait for recrawl and re-run URL Inspection on the homepage.
